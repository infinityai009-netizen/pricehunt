# PriceHunt Homepage Refresh

This project contains the refreshed homepage UI wired as the default homepage.

## Local run

```bash
npm install
npm run dev
```

## Netlify

If your repo contains this project in a subfolder, set **Base directory** to `pricehunt_refresh_deploy`.
If the repo contains only this project, leave **Base directory** blank.

Build command:
```bash
next build
```

Publish directory:
```bash
.next
```
